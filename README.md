# DeLorme Page Finder

A Progressive Web App (PWA) that uses your device's GPS to display which **DeLorme Atlas & Gazetteer** page you are currently located on. Designed for geocachers, hikers, and off-grid navigators, this tool works offline and supports automatic state detection.

## üöÄ Features

*   **Real-Time GPS Tracking:** Displays current Latitude/Longitude in DDM format (Degrees Decimal Minutes).
*   **Auto-Load Logic:** Automatically detects which US State you are in and loads the corresponding map grids.
*   **Offline Capable:** Can be installed as a native app on Android and iOS. Includes a "Download All Maps" feature to cache data before leaving cell service.
*   **Multi-Grid Support:** Can display multiple overlapping boundaries simultaneously.
*   **High Contrast UI:** Dark mode design for better visibility outdoors and battery saving.
